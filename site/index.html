Lead Enrichment & Outreach Pipeline

A resilient automation pipeline that discovers brands from creators’ paid-partnership posts on Instagram, turns them into verified, scored, ready-to-contact leads, and safely hands the qualified ones to cold outreach — syncing every step to a CRM funnel by webhook. Built so that no record is silently lost and no API budget is overspent, even when third-party services fail.

Everything below is a live, reproducible run: 1,409 simulated Instagram posts → 129 brands. No real data or paid services.

Built for reliability

The safeguards that let this run unattended against paid, flaky, rate-limited third-party APIs — each shown with what it actually did this run.

Quota gate

Credits are checked before every paid API call. The run spent only $14.08 and pauses cleanly instead of overspending if credits run low.

Retry with backoff

Transient failures are retried with exponential backoff — 39 recovered this run. Permanent errors are never retried.

Dead-letter queue

Every failed record is captured with a reason — 21 this run — so failures are triaged, never silently dropped.

Dedup registry

Idempotent by design: the same brand is never processed or contacted twice. 5 duplicate(s) caught this run.

Rate limiter

Respects a daily send cap; overflow is deferred to the next run rather than dropped — 18 deferred this run.

Config-driven

Credit limits, retry counts, the daily cap, scoring thresholds, and the CRM webhook URL all live in one config file — nothing is hard-coded in the logic.
