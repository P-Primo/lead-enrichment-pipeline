Lead Enrichment & Outreach Pipeline

A resilient automation pipeline that turns a raw list of companies into verified, scored, ready-to-contact leads — then safely hands the qualified ones to cold outreach. It is built so that no record is silently lost and no API budget is overspent, even when third-party services fail or run out of credit.

Everything below is a live, reproducible run on 129 synthetic companies — no real data or paid services involved.

Built for reliability

The safeguards that let this run unattended against paid, flaky, rate-limited third-party APIs — each shown with what it actually did this run.

Quota gate

Credits are checked before every paid API call. The run spent only $14.08 and stops cleanly instead of overspending if credits run low.

Retry with backoff

Transient failures (timeouts, rate limits) are retried with exponential backoff — 39 recovered this run. Permanent errors are never retried.

Dead-letter queue

Every failed record is captured with a reason — 21 this run — so failures are triaged, never silently dropped.

Dedup registry

Idempotent by design: the same company is never processed or contacted twice. 5 duplicate(s) caught this run.

Rate limiter

Respects a daily send cap; overflow is deferred to the next run rather than dropped — 18 deferred this run.

Config-driven

Credit limits, retry counts, the daily cap, and scoring thresholds all live in one config file — nothing is hard-coded in the logic.
